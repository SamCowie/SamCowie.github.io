Visualising Search Algorithms with 8 puzzle

Background

This application was developed by Sam Cowie for the University of Liverpool Computer Science Department
This applcation is intended as a learning aid to teach students about different search algorithms from COMP219 - Artificial Intelligence

The algorithms included in this application are:

  • Breadth First Search
  • Depth First Search
  • Uniform Cost Search
  • Greedy Search
  • A* Search

    • To use the app use the search panel to select an algorithm. You can then select to run the algorithm on the puzzle or learn more.
    Once the algorithm has solved the configuration of the 8 puzzle you can the view statistics from the search and compare the efficency of the different
    algorithms using the comparison table. For more information about the different statistics hover the mouse cursor on the "?" round the page.
